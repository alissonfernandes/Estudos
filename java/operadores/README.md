# Operadores
São símbolos especiais quais são capazes de realizar ações específicas em um, dois ou mais operadores e, em seguida, retornar um resultado.
 ## Operadores Aritméticos
 | Símbolo | Operação |
 |--- |--- |
 | + | Soma|
 |-|Subtração|
 |*|Multiplicação|
 |/|Divisão|
 |%|Resto da divisão|
