# Lista duplamente encadeadas
 As listas duplamente encadeadas ou lista duplamente liagadas, constituem uma variação das listas ligadas.

 A grande diferença das listas duplamente ligadas para as listas simples é que elas são bidirecionais.
 Vemos que, na **Lista Encadeada** naturalmente, não conseguimos "andar para trás", pois os _nós_ de uma lista ligada conhecem somente quem é o próximo
 elemento da lista. Já nas Listas Duplamente Encadeadas, os _nós_ conhecem quem é o próximo elemento e também quem é o elemento anterior, o que permite a
 navegação reversa.
