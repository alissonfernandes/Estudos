Como criar degradê com CSS

Exemplo de código

        
        *{
            height: 100%;
        }
        body{
            background-image: linear-gradient(to bottom, #0DE0D6, #0AADA5, #06615C);
            background-attachment: fixed;
        }

Propriedade background-image

  • linear-gradient

    • to right
    • to left
    • to top
    • to bottom
    • circle

  • radial-gradient

    • circle
