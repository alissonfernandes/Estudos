Imagens que se adaptam sozinhas

Exemplo de código

        
            <picture>
                <source media="(max-width: 750px)" srcset="../ex016/img/foto-p.png" type="image/png">
                <source media="(max-width: 1050px)" srcset="../ex016/img/foto-m.png" type="image/png">
                <img src="../ex016/img/foto-g.png" alt="Imagem flexível">
            </picture>

Parâmetros: <source>

  • media

    Indica o tamanho máximo ou mínimo a ser considerado para carregar a imagem indicada no atributo srcset

    • Exemplo para valores: media="(max-width: 750px)
    • Exemplo para valores: media="(max-width: 1050px)
  • srcset

    Vai configurar o nome/diretório da imagem que será carregada quando o tamanho indicado for atingido.

    • Exemplo: srcset="photo.png"
  • type

    Vai indicar o media type da imagem que usamos

    • image/png
    • image/jpeg
    • Para ver mais: IANA.org
