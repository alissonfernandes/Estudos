Parâmetros: <video>

  • src

    Diretório onde esta localizado o arquivo no qual deseja exibir.

  • controls

    Adiciona controladores

  • poster

    Adiciona uma imagem de fundo de vídeo antes de reproduzi-lo. Para isso você deve informar o caminho da imagem

Parâmetros: <source>

  • src

    Diretório onde esta localizado o arquivo no qual deseja exibir.
