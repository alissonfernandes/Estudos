Colocando áudio no seu site

Exemplo de código

        
            <audio src="media/music2.mp3" controls autoplay></audio>
            <br>
            <audio preload="metadata" controls autoplay loop>
                <source src="media/music2.mp3" type="audio/mp3">
                <source src="media/music.wma" type="audio/wma">
                <p>
                    Infelizmente seu navegador não consegue reproduzir áudio, <a href="media/music2.mp3">clique aqui para baixar o arquivo</a>
                </p>
            </audio>

Parâmetros: <audio>

  • src

    Diretório onde esta localizado o arquivo

  • controls

    Adiciona controladores

  • autoplay

    Auto executar o áudio

  • preload:

    Modo de execução de áudio

    • auto
    • metadata (recomendado)
    • none

    Diretório onde esta localizado o arquivo
