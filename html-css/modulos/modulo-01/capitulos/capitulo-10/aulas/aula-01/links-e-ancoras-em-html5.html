Links e Âncoras em HTML5

  • <a>

    Tag utilizada para criar âncoras, para criar hyperlinks, que permitem que você ligue um ponto a outro na World Wid Web. Ex.:
    <a href="" target="_blank" rel="external">

    • Parâmetro: target=""

      • Valor: _blank
        • Vai abrir o link em uma nova janela em branco
      • Valor: _self
        • Vai abrir o link na janela ou frama atual (padão)
      • Valor: _top
        • Vai desfazer todos os frames e abrir o destino no navegador completo
      • Valor: _parent
        • Similar ao uso do _top em uma referência à janela mãe

    • Parâmetro: rel=""

      • Valor: next
        • Indica que o link é para a próxima parte do documento atual
      • Valor: prev
        • Indica que o link é para a parte anterior do documento atual
      • Valor: author
        • Indica que é um link para um site do autor do artigo atual
      • Valor: external
        • Indica que é um link para outro site que não faz parte do site atual
      • Valor: nofollow
        • Indica que é um link para um site não endossado, como um link pago
