Principais formatações

Negrito / Destacação

Nessa frase temos um termo em negrito usando a tag <strong>.

Itálico

Nesta frase, temos um termo em itálico usando a tag <em>.

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>.

Texto grande e pequeno

Estamos criando um texto pequeno nesse parágrafo usando a tag <small>.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado usando a tag <del>.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase a indicar que ele foi adicionado depois. Para isso usamos a tag <ins>

Texo sobrescrito

Para inserir coisas do tipo X 20+3. Usamos a tag <sup>

Texto subscrito

Para adicionar coisas do tipo H2O. Usamos a tag <sub>
