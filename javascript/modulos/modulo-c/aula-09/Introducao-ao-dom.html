Introdução ao DOM

O que é DOM?

É um acrônimo para Document Object Model (Modelo de Objetos para Documentos). Ele está presente quando você roda JavaScript dentro do seu navegador por isso, o DOM não funciona dentro do nodeJs, por exemplo.

Árvore DOM

Tudo, dentro do JavaScript, está dentro de uma raiz chamada window e dentro do window possui outros três principais objetos: location document history

location

Diz qual é a localização do seu site, qual é a URL, qual é a página atual e a página anterior.

document

Dentro desse documento possui outro obejeto muito importante que é o html e dentro do objeto html pussui outros dois objetos: head e body

hitory

É um histórico que guarda de onde você veio e para onde você vai

window.document.chaset - mostra o set de caractéres
window.navigator.appName - mostra o nome do navegador atual
window.document.URL - mostra a url da página

Tipos de seletores

  • por marca -> getElementsByTagName()
  • por id -> getElementById()
  • por nome -> getElementsByName()
  • por classe -> getElementsByClassName()
  • por Seletor -> querySelector('div#p')
