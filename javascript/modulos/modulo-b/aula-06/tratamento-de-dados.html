Tratamento de dados

Conversão de dados

Number.parseFloat - Converte uma string ou inteiro para real
Number.parseInt - Converte uma string ou float para inteiro
Number - converte uma string para número
String - converte número para string
n.toString - possui a mesma funcionalidade do String: converte número para string

Formatação de string

s.toUpperCase - coloca todas as letras em maiúsculas
s.toLowerCase - coloca todas as letras em minúsculas
s.length - mostra quantos caracteres tem a string

Formatação de números

n1.toFixed(2) - formata um número para duas casas decimais n1.replace('.', ',') - troca uma string pela outra n1.toLocaleString('pt-BR', {style: 'currency', currency: 'BRL'}) - formata um número para valor monetário
