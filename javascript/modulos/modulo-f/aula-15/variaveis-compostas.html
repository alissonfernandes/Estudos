Variáveis simples

Variáveis simples só conseguem armazenar um valor por vez.

Variáveis compostas

Variáveis compostas devem ser capazes de armazenar vários valores em uma mesma estrutura.

push() - adiciona um elemento no array
sort() - ordena uma array
indexOf() - retorna a chave do valor do elemento passado como parâmetro
